Converting Data in Relational Databases to RDF

If you have data in a relational database, you can convert some or all of it to RDF using a tool called D2R Map. D2R Map is an Open Source, relational database to RDF mapping tool that allows you to connect to a relational database, extract data via an SQL query and, with knowledge of the relational schema, convert that data into RDF.

Once you have RDF, use the iTQLTM load command to load it into a Mulgara model. See Loading Existing RDF Data.
