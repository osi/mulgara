iTQL Commands

iTQLTM commands are used to query and update KowariTM databases. All commands are terminated with a semicolon. Where URIs occur as command parameters, they are always absolute and enclosed within angle brackets (< and >). For example, <rdf:type> and <http://site.domain.net/index.html>.
