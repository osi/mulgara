TKS Client Support

TKS provides a client-side JRDF graph interface for inserting, querying and removing triples. This provides a scalable RDF solution for remote client applications.

Client-side JRDF graphs can be used in conjunction with memory JRDF graphs. Memory implemented nodes and triples can be inserted into, removed from and used to query client-side JRDF graphs. Other JRDF implementations can also be used but may not be supported.

TKS also allows you to create read-only JRDF graphs from iTQL queries. This provides a powerful mechanism for displaying search results as a subgraph.
