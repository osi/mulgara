Version 1.1.0 Pre-release 1 (Build 1.1.0.192)

This release includes a major modification to the Kowari system architecture, known as Resolvers. Resolvers give Kowari the capability to query any type of data source by treating the data source as a graph. Also included in this release are features for improved performance plus support for XML Literal, XSD and unknown datatypes.

For more information on changes between releases, consult the revision history.

The downloads available include:

  • Complete Source
  • Kowari Lite Binaries
