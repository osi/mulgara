Mulgara Documentation

Two specific tools available for use with MulgaraTM are the Mulgara ViewerTM and Descriptors.

The Mulgara Viewer is a web-based query interface for Mulgara. It allows you to execute iTQLTM commands and then view the results in formatted HTML.

Descriptors are XSL stylesheets that perform specific, well-defined tasks in Mulgara. They are an alternative to using APIs. Some examples of useful tasks that Descriptors can be used for include:

  • Hiding complex Mulgara queries behind a simple interface.
  • Aggregating complex data by Descriptors calling other Descriptors.
  • Developing HTML front ends to Mulgara queries.
