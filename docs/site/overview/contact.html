Contact Information

 

Support

Mulgara is supported through the Mulgara community, specifically on the mulgara-general mailing list. All support requests for this open source project should first go through this list.

 

Mailing Lists

Discussions on Mulgara are held on the kowari-general mailing list. Please do not send HTML, rich text, or stylized mail to the list. If you use Mail, Outlook, Outlook Express or Eudora, you may be sending HTML mail by default. There is usually a setting that will allow you to send plain text mail instead.

 

Bug Reports and Feature Requests

Bug reports and features requests can be made on the Mulgara Project issue tracker.

 

Developers

Contact information for Mulgara developers can be found on the Mulgara developers list.
