HTML and CSS

This site uses web standards and as such its markup and layout should validate. It uses XHTML 1.0 Strict and follows the Appendix C. HTML Compatibility Guidelines. If this site does not render correctly with your browser, it may be that it does not support CSS2. If it does and you have problems, or you think someting could be improved, please get in touch.

For the interested, the original layout for this site comes from glish.com and bluerobot.com. It uses CSS3 selectors and may not render well in some browsers. As the current CSS validator only checks for compliance with CSS1 and CSS2, it will report "errors" in the style sheets. They should however, be 100% valid per W3C recommendations.
