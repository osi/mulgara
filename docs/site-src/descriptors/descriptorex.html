Descriptor Examples

The following three examples walk through the steps on how to develop Descriptors:

  1. Hello World. A simple Descriptor.
  2. List Models. List the models in a Mulgara database.
  3. VCard. Uses several Descriptors to generate a HTML page listing the VCards in a specified Mulgara model

Note - Knowledge of the W3C Extensible Stylesheet Language (XSL) is strongly recommended when developing Descriptors. Using Descriptors does not require any XSL knowledge.
