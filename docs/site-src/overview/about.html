About the Mulgara Project

MulgaraTM is an open source, massively scalable, transaction safe, purpose-built database for the storage of metadata. Mulgara commenced in July 2006 as a fork of the Kowari project. Kowari was released under an open source license in January 2004.

More information on Mulgara can be found in the Mulgara Overview.
