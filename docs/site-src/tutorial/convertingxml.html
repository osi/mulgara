Converting XML Data to RDF

If you have data in XML format (for example, from an entity extraction tool), you can convert some or all of it to RDF using the XML Stylesheet Language Transformations (XSLT) language. A list of XSLT Tools is available, as is a good XSLT Tutorial.

Once you have RDF, use the iTQLTM load command to load it into a Mulgara model. See the Loading Existing RDF Data section for more information.
